### 说明

通过添加参数可以指定浏览器环境,如上所述指定的为Chrome下运行

- -Dbrowser=firefox
- -Dbrowser=chrome
- -Dbrowser=ie
- -Dbrowser=edge
- -Dbrowser=opera
